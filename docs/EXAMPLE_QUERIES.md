## üí° Tips for Better Queries

1. **Be Specific**: Include names, dates, and filters
2. **Use Natural Language**: Ask questions as you would to a colleague
3. **Build on Previous Questions**: Use follow-ups for refinement
4. **Include Context**: Mention relevant details
5. **Try Different Phrasings**: If one doesn't work, rephrase
