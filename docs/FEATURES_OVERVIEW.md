# Features Overview

Complete feature list for YNM Safety Price Engine & CRM System.

## Price Engine Features

### MBCB (Metal Beam Crash Barriers)
- ✅ W-Beam calculation
- ✅ Thrie Beam calculation
- ✅ Double W-Beam calculation
- ✅ Material cost calculation
- ✅ Transportation cost
- ✅ Installation cost
- ✅ Fastener options (Hex bolts, Button bolts)
- ✅ PDF generation
- ✅ Save quotations

### Signages
- ✅ Reflective signage calculation
- ✅ MS (Mild Steel) component calculation
- ✅ Board specifications
- ✅ MS angle and pipe options
- ✅ Area-based pricing
- ✅ Combined pricing
- ✅ PDF generation

### Paint
- ✅ Thermoplastic paint calculation
- ✅ Area-based calculation
- ✅ Cost per square foot
- ✅ Profit margin calculation
- ✅ PDF generation
